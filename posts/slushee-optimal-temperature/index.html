Have you ever reached the end of an Icee or Slurpee only to find a desolate, dry block of white ice? This is the “Slushee Dead Zone”—a state where the liquid-to-solid ratio has collapsed, leaving the remaining ice stranded without enough syrup to be drinkable.

The problem is one of phase-change synchronization. For a perfect drinking experience, the rate at which you consume the liquid must match the rate at which the ambient heat melts the ice.

1. Geometry: The Tapered Frustum

7-11 cups are not perfect cylinders; they are frustums (tapered cones).

2. Thermodynamics: Continuous Phase Change

Instead of modeling discrete sips, we can model the slushee as a continuous system.

Heat Transfer

Heat flows from the ambient air (\(T_{amb}\)) through the cup walls: \[\frac{dQ}{dt} = h \cdot A \cdot (T_{amb} - 0^\circ C)\] This heat melts the ice at a rate of \(\dot{m}_{melt} = \frac{1}{L_f} \frac{dQ}{dt}\).

Consumption

We assume a constant volumetric consumption rate \(C\) (oz/min). Because of the straw’s liquid bias, the rate of ice removal \(\dot{m}_{ice\_out}\) is lower than the rate of liquid removal.

3. The Master Differential Equation

By combining the geometry of the frustum with the laws of thermodynamics, we can derive a single differential equation that governs the Ice Ratio (\(R\)) over time.

Let \(V(t)\) be the current volume of the slushee. The rate of change of the ice mass \(m_i\) is driven by two competing forces: melting and drinking.

\[\frac{dm_i}{dt} = -\underbrace{\frac{h \cdot A(V) \cdot T_{amb}}{L_f}}_{\text{Melting}} - \underbrace{C \cdot \rho_i \cdot (\beta R)}_{\text{Drinking}}\]

Where: - \(A(V)\) is the surface area as a function of volume for a frustum. - \(\beta\) is the liquid bias factor (the “Ice-in-Straw Bias”). - \(C\) is the consumption rate.

The Final Expanded Solution for the change in ice ratio \(\frac{dR}{dt}\) is:

\[\frac{dR}{dt} = \frac{1}{M} \left[ \dot{m}_i (1 - R) - R \dot{m}_l \right]\]

Plugging everything in:

\[\frac{dR}{dt} = -\frac{h \cdot A(V) \cdot T_{amb}}{M \cdot L_f} - \frac{C}{M} \left[ \rho_i \beta R (1-R) - R \rho_l (1 - \beta R) \right]\]

This equation reveals the “Critical Temperature”: the point where the melting term (which decreases \(R\)) perfectly cancels out the drinking term (which increases \(R\) due to liquid bias).

Code 
ozToMl = 29.57
inchToCm = 2.54
Lf = 334 // J/g
h_coeff = 9.5 // W/m^2K
rho_ice = 0.917 // g/cm^3
rho_liq = 1.0 // g/cm^3

cupSizes = ({
  small: { h: 4.5 * inchToCm, r1: (2.25/2) * inchToCm, r2: (3.25/2) * inchToCm },
  medium: { h: 6.0 * inchToCm, r1: (2.5/2) * inchToCm, r2: (3.75/2) * inchToCm },
  large: { h: 7.0 * inchToCm, r1: (2.75/2) * inchToCm, r2: (4.25/2) * inchToCm },
  xlarge: { h: 8.5 * inchToCm, r1: (3.0/2) * inchToCm, r2: (4.75/2) * inchToCm }
})

results = {
  const d = cupSizes[cupSize];
  const dt = 0.1; // minutes
  const C_ml = consumptionRate * ozToMl;
  
  let m_ice = (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)) * initialIceRatio * rho_ice;
  let m_liq = (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)) * (1 - initialIceRatio) * rho_liq;
  
  const history = [];
  let t = 0;

  while (m_ice + m_liq > 1 && history.length < 1000) {
    let currentVol = (m_ice / rho_ice) + (m_liq / rho_liq);
    let iceRatio = m_ice / (m_ice + m_liq);
    
    // Geometry
    let heightFrac = Math.max(0, currentVol / (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)));
    let y = d.h * heightFrac;
    let ry = d.r1 + (d.r2 - d.r1) * heightFrac;
    let Area = Math.PI * (d.r1 + ry) * Math.sqrt((ry - d.r1)**2 + y**2) + Math.PI * d.r1**2;
    Area = Area / 10000;

    history.push({ time: t, iceRatio: iceRatio * 100, vol: currentVol / ozToMl });

    // Thermodynamics (Heat in)
    let dQ = h_coeff * Area * (ambientTempC - 0) * (dt * 60);
    let melted = dQ / Lf;
    melted = Math.min(melted, m_ice);
    m_ice -= melted;
    m_liq += melted;

    // Consumption (Mass out)
    let volOut = C_ml * dt;
    let r_out = iceRatio * liquidBias;
    m_ice -= Math.min(m_ice, volOut * r_out * rho_ice);
    m_liq -= Math.min(m_liq, volOut * (1 - r_out) * rho_liq);

    t += dt;
  }
  return history;
}

Summary of Optimal Temperatures

The following table shows the minimum ambient temperature required to finish a 7-11 slushee at a steady consumption rate of 2 oz/min without hitting the Dead Zone.

Cup Size Min Ambient Temp
Small (12 oz) ~14°C (57°F)
Medium (22 oz) ~21°C (70°F)
Large (30 oz) ~26°C (79°F)
X-Large (40 oz) ~32°C (90°F)

Conclusion

The larger the cup, the lower the surface-area-to-volume ratio, and thus the slower the relative melting rate. If you’re going for the 40oz X-Large, you’d better be drinking it in the heat of mid-July, or prepare for a very icy ending.
