Have you ever reached the end of an Icee or Slurpee only to find a desolate, dry block of white ice? This is the “Slushee Dead Zone”—a state where the liquid-to-solid ratio has collapsed, leaving the remaining ice stranded without enough syrup to be drinkable.

The problem is one of phase-change synchronization. For a perfect drinking experience, the rate at which you consume the liquid must match the rate at which the ambient heat melts the ice.

1. Geometry: The Tapered Frustum

7-11 cups are not perfect cylinders; they are frustums (tapered cones). This affects the surface area \(A\) exposed to the air as the volume decreases.

Mathematical Formulas

To model the heat transfer accurately, we use the following formulas for a frustum of height \(h\), base radius \(r_1\), and top radius \(r_2\):

Volume (\(V\)): \[V = \frac{\pi h}{3} (r_1^2 + r_1 r_2 + r_2^2)\]

Lateral Surface Area (\(A_{lat}\)): The surface area of the sides (where most heat transfer occurs) is: \[A_{lat} = \pi (r_1 + r_2) \sqrt{(r_2 - r_1)^2 + h^2}\]

Total Surface Area (\(A_{total}\)): Including the base (ignoring the lid): \[A_{total} = A_{lat} + \pi r_1^2\]

2. Thermodynamics: Heat Transfer & Phase Change

Heat flows from the ambient air (\(T_{amb}\)) through the cup walls to melt the ice: \[\frac{dQ}{dt} = h_{coeff} \cdot A \cdot (T_{amb} - 0^\circ C)\]

When heat enters, it melts a mass of ice \(m_{melt}\): \[m_{melt} = \frac{dQ}{L_f}\] Where \(L_f = 334\ J/g\) is the latent heat of fusion.

Consumption Bias

When you drink through a straw, the liquid is pulled more easily than the ice crystals. \[\text{Ice Ratio in Sip} = 0.5 \times \text{Current Ice Ratio in Cup}\]

ozToMl = 29.57
inchToCm = 2.54
Lf = 334 // J/g
h_coeff = 9.5 // W/m^2K
rho_ice = 0.917 // g/cm^3
rho_liq = 1.0 // g/cm^3

// Cup Geometry Data
cupSizes = ({
  small: { h: 4.5 * inchToCm, r1: (2.25/2) * inchToCm, r2: (3.25/2) * inchToCm },
  medium: { h: 6.0 * inchToCm, r1: (2.5/2) * inchToCm, r2: (3.75/2) * inchToCm },
  large: { h: 7.0 * inchToCm, r1: (2.75/2) * inchToCm, r2: (4.25/2) * inchToCm },
  xlarge: { h: 8.5 * inchToCm, r1: (3.0/2) * inchToCm, r2: (4.75/2) * inchToCm }
})

function simulate(temp, sizeKey, interval, initRatio, sipVol) {
  const d = cupSizes[sizeKey];

  let m_ice = (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)) * initRatio * rho_ice;
  let m_liq = (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)) * (1 - initRatio) * rho_liq;
  
  const history = [];
  let t = 0;
  const v_sip = sipVol * ozToMl;

  while (m_ice + m_liq > v_sip * 0.1 && history.length < 500) {
    let currentVol = (m_ice / rho_ice) + (m_liq / rho_liq);
    let iceRatio = m_ice / (m_ice + m_liq);
    
    let heightFrac = Math.max(0, currentVol / (d.h * Math.PI / 3 * (d.r1**2 + d.r1*d.r2 + d.r2**2)));
    let y = d.h * heightFrac;
    let ry = d.r1 + (d.r2 - d.r1) * heightFrac;
    let Area = Math.PI * (d.r1 + ry) * Math.sqrt((ry - d.r1)**2 + y**2) + Math.PI * d.r1**2;
    Area = Area / 10000; // cm2 to m2

    history.push({ 
      time: t / 60, 
      iceRatio: iceRatio * 100, 
      vol: currentVol / ozToMl 
    });

    let Q = h_coeff * Area * (temp - 0) * interval;
    let melted = Q / Lf;
    melted = Math.min(melted, m_ice);
    m_ice -= melted;
    m_liq += melted;

    let r_sip = iceRatio * 0.5;
    m_ice -= Math.min(m_ice, v_sip * r_sip * rho_ice);
    m_liq -= Math.min(m_liq, v_sip * (1 - r_sip) * rho_liq);
    t += interval;
  }
  return history;
}

4. Final Summary: Optimal Temperatures

The “Optimal Temperature” is defined as the minimum ambient temperature required to ensure the ice ratio never exceeds 90% during consumption.

Assume 2oz sips and an initial ice ratio of 60%.

Cup Size Leisurely (1 sip/min) Fast (1 sip/30s)
Small (12 oz) ~14°C (57°F) ~26°C (79°F)
Medium (22 oz) ~21°C (70°F) ~34°C (93°F)
Large (30 oz) ~26°C (79°F) ~40°C (104°F)
X-Large (40 oz) ~31°C (88°F) ~46°C (115°F)

Conclusion

If you are an extra-large slushee drinker who finishes their drink in 10 minutes (fast), you effectively need to be standing in a heat wave (46°C) to keep the ice drinkable. For the rest of us, the 22oz “Medium” in a comfortable 21°C room remains the golden standard of phase-change synchronization.
